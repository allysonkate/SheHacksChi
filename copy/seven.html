Tumblr Templates

You’ve probably noticed some code in Tumblr wrapped in curly braces, or “mustaches”, like this: {title}.

Code that is wrapped in these mustaches is a placeholder for information you’ve already stored in Tumblr. Remember when you made a title for your blog? We named ours “She Hacks Tumblr”. So {title} will display the information “She Hacks Tumblr”. The same goes for {description}. Anywhere you place this code, it will output your description you filled in.
